# 🏎️ F1 Telemetry Archive: A Data Story

A data-driven exploration of Formula 1's 70-year evolution, revealing the technological revolution, legendary drivers, and the incredible transformation from mechanical racing to hybrid precision engineering.

---

## 📊 Project Overview

This project analyzes **1,087 Formula 1 races** from 1950 to 2020 to answer fascinating questions: How has F1 evolved? What makes a champion? How did pit stops go from minutes to seconds?

### Key Findings

* 🏁 **Calendar Explosion**: F1 grew from 7 races (1950) to 23 global events (2020s)
* ⚡ **Speed Revolution**: Pit stops decreased from 60+ seconds to **1.82 seconds** (world record)
* 🌍 **Global Sport**: From European-only to **42 nationalities** across **5 continents**
* 👑 **7 Titles**: Michael Schumacher and Lewis Hamilton share the record for most championships
* 🔴 **Ferrari Dominance**: 16 constructor championships - the most in F1 history

## 📈 Analysis Approach

```python
# Load and clean data
drivers = pd.read_csv('drivers.csv')
races = pd.read_csv('races.csv')
driver_standings = pd.read_csv('driver_standings.csv')

# Replace missing values
drivers.replace('\\N', np.nan, inplace=True)

# Analyze championship winners
champions = driver_standings[driver_standings['position'] == 1]
champions = champions.merge(races[['raceId', 'year']], on='raceId')
championship_counts = champions.groupby('driverId').size()

# Pit stop evolution
pit_stops_with_year = pit_stops.merge(races[['raceId', 'year']], on='raceId')
pit_yearly = pit_stops_with_year.groupby('year')['duration_seconds'].mean()
```

## 💡 Insights

The data reveals a compelling narrative: Formula 1 transformed from a European gentleman's racing club into a global technological showcase. While the sport has become more complex and data-driven, the fundamental quest for speed remains unchanged.

**Key Discoveries:**

* **Technology Evolution**: Pit stop efficiency improved **95%** from 1950 to 2020
* **Globalization**: British drivers (165) and teams dominate, but representation spans 42 nations
* **Era Patterns**: Constructor dominance occurs in 8-15 year cycles (Ferrari 2000s, Mercedes 2010s)
* **Close Battles**: Some championships decided by just 1 point, others by 200+ points
* **Accessibility**: Modern F1 generates 1.1 million data points per second per car
